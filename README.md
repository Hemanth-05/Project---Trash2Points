# Trash2Points

Trash2Points is a gamified recycling habit app for apartment residents. It encourages sustainable behavior through points, streaks, and community competition.

## App Idea

Trash2Points is a frontend-only hackathon prototype focused on behavior change rather than logistics. Residents log their recycling actions, earn points by material type, maintain a daily streak, and compare progress on a lightweight apartment-community leaderboard.

## Features

- Welcome and join flow with localStorage persistence
- Recycling action logging for Plastic, Paper, Glass, and Organic items
- Points system with category-based scoring
- Simple daily streak logic for consistent recycling habits
- Activity feed with timestamps
- Badge progression from Beginner Recycler to Sustainability Hero
- Demo leaderboard combining dummy residents with the active user
- Community impact card with hackathon-friendly sample metrics
- Reset demo button to clear local state and restart the experience
- Responsive, polished UI for desktop and mobile browser demos

## Tech Stack

- React
- Vite
- Plain CSS
- localStorage for persistence

## How To Run

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.
